# Typescript Node Js API Postgres TypeORM Swagger

NESTJs - A Typescript/ Javascript equivalent of Spring boot.

Opioniated very similar to Angular. Modular

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## PostGres

Postgres commands

### Connect locally

psql -h localhost

### Create Database

create database <dbName>

\c <dbName>

### Create user

create user root with encrypted password 'root';

### Grant priveleges to a user

grant all privileges on database photos to root;

### List all databases

\l

### List all tables

\dt
